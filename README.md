# Nano Botball Library

Nano is a C++ library for Botball teams. It wraps and extends the
[libkipr/libwallaby](https://github.com/kipr/libwallaby) and provides
**thread-safe** functions as well as extra helper functions and classes.

Nano is named after the robot in Nichijou.
